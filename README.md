# TarotToday 🔮

매일 타로 카드로 오늘의 운세를 확인하세요. 78장의 라이더-웨이트-스미스(RWS) 타로 덱을 기반으로 한 신비로운 운세 앱입니다.

![TarotToday](https://img.shields.io/badge/Tarot-Daily%20Fortune-purple)
![React](https://img.shields.io/badge/React-19.2-blue)
![Vite](https://img.shields.io/badge/Vite-8.0-orange)

## 🌟 주요 기능

- **일일 타로 리딩**: 매일 무작위로 선택된 카드로 운세를 확인하세요
- **정방향/역방향 해석**: 카드의 방향에 따른 다양한 해석 제공
- **한국어 번역**: 모든 카드 이름, 키워드, 해석이 완벽하게 한국어로 제공됩니다
- **부드러운 애니메이션**: Framer Motion으로 구현한 카드 뒤집기 효과
- **행운 로또 번호**: 카드와 함께 오늘의 행운 번호 3개도 받아보세요
- **공유 기능**: 결과를 이미지로 저장하거나 SNS에 공유할 수 있습니다
- **반응형 디자인**: 모바일과 데스크톱 모두에서 완벽하게 작동

## 🎴 타로 카드

- **메이저 아르카나** (22장): 광대에서 세계까지, 삶의 큰 주제를 다루는 카드들
- **마이너 아르카나** (56장): 완드, 컵, 소드, 펜타클 4개 슈트의 일상적인 의미

## 🛠️ 기술 스택

- **Frontend**: React 19.2, Vite 8.0
- **Styling**: Tailwind CSS 4.2
- **Animation**: Framer Motion 12.38
- **Icons**: React Icons 5.6
- **Build**: Vite

## 📦 설치 및 실행

### 로컬에서 실행하기

```bash
# 레포지토리 클론
git clone https://github.com/tramper2/TaroToday.git
cd TaroToday/Source

# 의존성 설치
npm install

# 개발 서버 시작
npm run dev
```

http://localhost:3000/TarotToday/ 에서 접속하세요.

### 빌드

```bash
npm run build
```

### 배포

```bash
npm run deploy
```
